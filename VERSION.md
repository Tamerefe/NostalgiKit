RELEASE NOTES
=============

Version 1.0.0 (July 22, 2025)
- Initial release of NostalgiKit
- Three complete retro games included:
  * Number Oracle (Card Guess Game)
  * War Game (Turn-Based Combat)
  * River Puzzle (Logic Challenge)
- Authentic vintage handheld gaming interface
- Full keyboard control support
- Cross-platform compatibility (Windows, macOS, Linux)
- Complete MIT licensing and copyright protection

SYSTEM REQUIREMENTS
===================
- Python 3.7 or higher
- tkinter (usually included with Python)
- Pillow (PIL) for image processing
- Minimum 4MB free disk space
- 256MB RAM recommended

SUPPORTED PLATFORMS
===================
- Windows 10/11
- macOS 10.14+
- Linux (Ubuntu 18.04+, Fedora 30+, etc.)

For technical support and updates, visit the project repository.
